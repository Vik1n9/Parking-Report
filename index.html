✨ 混合輸入模式 (支援簡稱)
• 您可以使用順序值 + 鍵值對混合輸入。順序值依序對應：一樓以上、一樓以下、P1、P3、紡A、紡B、紡C、紡D、紡E、柏油路。
• 鍵值對可使用完整名稱或簡稱：
  A = 紡A, B = 紡B, C = 紡C, D = 紡D, E = 紡E, R = 柏油路, P1/P3 不變。
• 範例：423 256 7 7 E=0.4
  → 一樓以上=423, 一樓以下=256, P1=7, P3=7, 紡E=0.4 (尚有4台車)，其餘區域設為「x」(未停車)。
• 若只輸入順序值（無等號），即傳統順序模式；若只輸入鍵值對（無純數字），即純跳項模式；若混合，則先依序填入，再用鍵值對覆蓋。
✅ 數字格式：車位區直接寫數字 (如 120)、成數區寫整數 (如 8)、台車寫 0.1~0.9、全滿寫 0、沒停車寫 x。
📌 中控回報台車數字直接顯示阿拉伯數字 (例如「尚有4台車」)，時間格式為 HH:MM。
