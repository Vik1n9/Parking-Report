🚗 停車回報轉換器 (跳項/順序 雙模式)

⚡ 目前解析模式: 順序模式 (空格順序)
🚗 車位區: 數字=車位數 🅿️ 成數區: 8 → 8成 🔢 0.1~0.9 → X台車 ❌ x → 沒停車 🈵 0 → 全滿
📢 中控回報格式 (口播/紀錄)
📎 Excel 備註欄 (智能轉換: 數字/幾成/未停車/幾台車)
✨ 雙模式說明 (保留快速輸入 + 跳項輸入)
🚀 模式① 順序模式(快速)：直接輸入10個值，用空格分隔。少於10個自動補「x」，多於10個忽略。
🧠 模式② 跳項模式(靈活)：使用「區域名稱=數值」格式，多組用空格分開。
  👉 支援的區域名稱：一樓以上、一樓以下、P1、P3、紡A、紡B、紡C、紡D、紡E、柏油路
  👉 範例：一樓以上=120 P1=0.2 柏油路=6 → 未指定的區域自動設為「x」(沒停車)
✅ 數字格式：車位區直接寫數字 (如 120)、成數區寫整數 (如 8)、台車寫 0.1~0.9、全滿寫 0、沒停車寫 x。
